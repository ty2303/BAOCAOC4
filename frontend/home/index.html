Trang chủ

Frontend nhỏ này dùng để test nhanh backend. Bạn chỉ cần mở trang, nhập dữ liệu cơ bản và xem kết quả ngay trên trình duyệt.

Đi nhanh

Mở trang sản phẩm Mở trang giỏ hàng

Cách dùng

  • Chạy server bằng npm start.
  • Mở trang sản phẩm để gọi /api/v1/products.
  • Mở trang giỏ hàng và dán token USER để test API giỏ hàng.
